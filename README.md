A responsive, accessible web application that generates beautiful color palettes based on a seed color and a chosen color harmony mode. Perfect for designers and developers looking for quick color inspiration.

## üöÄ Features

- **Custom Base Color:** Choose any starting color using the native system color picker.
- **Multiple Color Modes:** Generate schemes using various harmonies (Monochrome, Analogic, Complementary, Triad, Quad, etc.).
- **One-Click Copy:** Click on any generated color strip to instantly copy its Hex code to your clipboard.
- **Toast Notifications:** Get clear, non-intrusive visual feedback when a color is copied or if an error occurs.
- **Fully Responsive:** Seamlessly adapts from full-screen mobile layouts to a sleek card-style interface on larger screens.
- **Built for Accessibility:** Includes keyboard navigation support, semantic HTML (`<button>` tags for interactive elements), and ARIA labels for screen readers.

### Installation

Clone the repository to your local machine:

```bash

cd color-scheme-generator
```

## Getting Started

Install the dependencies and run the project

```

```
